```

```


## rxn-img
### 🎯 核心能力
`rxn-img` 是一个**企业科技信息图风格的 AI 配图生成技能**。它定义了一套统一的视觉风格（等距透视、扁平微立体、科技蓝主色调），用于将复杂的技术概念转化为直观、专业的图形表达。

### 📌 什么场景调用
| 适用场景 | 示例 |
|---------|------|
| ✅ 演示文稿配图 | PPT 路演、技术分享、内部培训 |
| ✅ 文档/报告插图 | Word/PDF 报告、白皮书封面 |
| ✅ 博客/文章头图 | 技术博客、行业分析文章 |
| ✅ 产品营销素材 | SaaS 着陆页、功能介绍页 |

**不适用**：需要真实摄影感、高度艺术性品牌视觉、面向消费者的情感化营销。

### 📋 如何调用
**高效命中，只需：**
```
`Use Skill: rxn-img` + 描述你想要的配图内容
```

#### 📊 场景1：演示文稿配图
- "帮我生成一张关于**微服务架构**的 PPT 配图"
- "我需要一个**云原生技术栈**的信息图，用于技术分享"
- "生成一张**数据中台**架构示意图，放在路演 PPT 里"

#### 📄 文档/报告插图
- "帮我做一张**AI 大模型**的技术概念图，要放在报告里"
- "生成一张白皮书封面图，主题是**数字化转型**"
- "我需要一张**网络安全防护体系**的插图用于 PDF 报告"

#### 📝 博客/文章头图
- "帮我生成一篇关于**低代码平台**的博客头图"
- "需要一张**边缘计算**主题的文章封面图"

#### 🎯 产品营销素材
- "帮我做一个 SaaS 产品的**功能架构图**，风格要专业商务"
- "生成一张**智能客服系统**的产品介绍配图"
