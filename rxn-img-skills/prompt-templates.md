# Prompt 模板 — 企业科技信息图风格

## 使用说明

1. 根据需求选择合适的模板
2. 将 `{占位符}` 替换为实际内容
3. 根据需要调整色彩方案
4. **`[TEXT]` 区块中的文字必须使用中文（专业术语除外）**
5. 将生成的 prompt 传入图像生成工具

### 语言要求（重要）

所有模板均包含 `[LANGUAGE]` 区块，**必须保留此区块**。该区块确保生成的图片中：
- 标题、标签、说明、标注等文字一律使用**中文**
- 仅行业通用缩写/专业术语（如 AI、API、SaaS、HTML、CSS、JavaScript 等）可保留英文
- 中英文混排时，英文术语前后加空格，保持排版整洁

## 模板 1：通用基础模板

适用于任何企业科技信息图场景，作为起点进行定制。

```
[PURPOSE]: {用途描述，如"AI治理主题的PPT演示配图" / "技术博客文章头图"}

[LANGUAGE]: 图片中所有文字使用中文，仅专业术语（如 AI、API、SaaS 等）可保留英文

[STYLE]:
- 等距透视（isometric projection），30度视角
- 扁平化设计 + 微立体感，柔和阴影
- 企业科技风格，类似 Google Material Design / SaaS 产品营销图
- 纯白背景，大量留白
- 几何简化的人物和物体
- 无面部特征的简约人物
- 一致的细线条描边

[COLOR PALETTE]:
- 主色：科技蓝 (#3B82F6)
- 辅助色：珊瑚红 (#E74C3C)、翡翠绿 (#27AE60)、琥珀黄 (#F5A623)
- 中性色：冷灰色系 (#6B7280, #9CA3AF, #E5E7EB)
- 点缀：青色荧光 (#00D4FF) 发光效果

[COMPOSITION]:
- {构图类型：中心焦点 / 水平叙事流 / 多元素并列 / 层叠递进}
- {构图细节描述}
- 清晰的视觉层级
- 充足的留白空间

[CONTENT]:
- {核心元素描述}
- {辅助元素描述}
- {图标/符号描述}
- {视觉隐喻描述}

[TEXT]:
- 主标题：{中文标题}
- 副标题/标签：{中文副标题或标签}
- 元素标注：{各元素的中文标注，专业术语可保留英文}
- 说明文字：{中文说明}

[QUALITY]:
- 高清矢量感
- 干净的边缘，抗锯齿
- 专业企业级品质
- {比例要求：16:9 横版 / 4:3 / 1:1 方形}
```

## 模板 2：中心焦点构图模板

适用于封面图、核心概念展示、单一主题强调。

```
[PURPOSE]: {用途描述}

[STYLE]: 等距透视，3D渲染感，企业科技风格，纯白背景，柔和全局光照，几何简化元素

[COLOR PALETTE]:
- 主色：科技蓝 (#3B82F6) 占主导
- 辅助色：{根据主题选择 1-2 个辅助色}
- 中性色：冷灰色系

[COMPOSITION]: 中心焦点构图
- 核心元素居中放置，视觉重心明确
- 周围环绕支撑元素，形成放射状或对称布局
- 多层深度：前景（框架）→ 中景（核心）→ 背景（氛围元素）
- 充足留白

[CONTENT]:
- 中心：{核心元素描述，如"半透明蓝色发光大脑，可见神经通路"}
- 环绕：{周围元素描述，如"六个护栏柱，连接彩色围栏带"}
- 氛围：{背景氛围元素，如"漂浮的几何芯片，淡色数据字符串"}
- 基座：{底部支撑结构，如"多层方形平台，白色和蓝色交替"}

[TEXT]:
- 主标题：{中文标题，如"智能治理框架"}
- 副标题：{中文副标题，如"负责任的 AI 工程"}
- 核心标注：{中文标注}

[QUALITY]: 高清3D渲染感，柔和阴影，干净边缘，专业企业级品质，{比例}
```

## 模板 3：水平叙事流模板

适用于时间线、流程图、演进对比、阶段展示。

```
[PURPOSE]: {用途描述}

[LANGUAGE]: 图片中所有文字使用中文，仅专业术语（如 AI、API、SaaS 等）可保留英文

[STYLE]: 等距透视，扁平化设计+微立体感，企业科技风格，纯白背景，无面部特征简约人物

[COLOR PALETTE]:
- 从左到右色彩饱和度递增
- 左侧：冷灰色系（代表过去/传统）
- 中间：中性色调（代表过渡/当前）
- 右侧：高饱和主色（代表未来/理想）
- 辅助色：{根据阶段数量分配}

[COMPOSITION]: 从左到右的水平叙事流
- {N}个阶段等距排列
- 底部有连接路径（道路/箭头/流程线）
- 每个阶段有独立的视觉标识
- 人物/元素尺寸从左到右递增（强化推进感）

[CONTENT]:
- 阶段 1（左侧）：{描述，如"灰色调，简约人物骑灰马，代表传统方法"}
- 阶段 2（中间）：{描述，如"中性色调，人物与未来感终端交互，代表过渡阶段"}
- 阶段 3（右侧）：{描述，如"高饱和蓝色调，人物骑蓝色发光马，代表增强方案"}
- 连接元素：{描述，如"深色道路，白色虚线，强化行进感"}
- 背景元素：{描述，如"路标、护栏等基础设施元素"}

[TEXT]:
- 阶段 1 标题：{中文，如"传统方法"}
- 阶段 2 标题：{中文，如"工具化阶段"}
- 阶段 3 标题：{中文，如"增强方案"}
- 总标题：{中文总标题}

[QUALITY]: 高清矢量感，干净边缘，专业企业级品质，16:9 横版比例
```
