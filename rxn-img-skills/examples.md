# 示例参考 — 企业科技信息图风格

以下示例基于实际配图分析提炼，可作为生成新配图时的直接参考。

**语言要求**：所有示例中的图片文字均使用中文，仅专业术语保留英文。

---

## 示例 1：四柱概念图（多元素并列）

**来源**：Four Guardrails — AI 治理框架

**适用场景**：框架概述、多维度对比、原则展示

**完整 Prompt**：
```
[PURPOSE]: 企业科技信息图配图，用于AI治理主题的PPT演示

[LANGUAGE]: 图片中所有文字使用中文，仅专业术语（如 AI、API 等）可保留英文

[STYLE]: 等距透视，扁平化设计+微立体感，企业科技风格，纯白背景，几何简化元素，柔和阴影，一致的细线条描边

[COLOR PALETTE]: 四色编码系统 - 蓝色(#4A90E2)、红色(#E74C3C)、黄色(#F5A623)、绿色(#27AE60)，冷灰色基座(#E5E7EB)

[COMPOSITION]: 四个垂直柱体等距排列，从左到右均匀分布，每个柱体顶部有代表性图标，底部共享一个科技感基座（电路板纹理），等间距排列

[CONTENT]:
- 蓝色柱：顶部放置文档+放大镜图标，代表上下文工程和信息检索
- 红色柱：顶部放置盾牌+AI机器人头图标（带青色发光效果），代表架构约束和安全验证
- 黄色柱：顶部放置双箭头循环图标，代表反馈循环和迭代改进
- 绿色柱：顶部放置生长植物+循环箭头图标，代表熵管理和可持续性
- 共享基座：浅灰色平台，带电路板纹理图案

[TEXT]:
- 主标题："四大护栏"
- 蓝色柱标签："上下文工程"
- 红色柱标签："架构约束"
- 黄色柱标签："反馈循环"
- 绿色柱标签："熵管理"

[QUALITY]: 高清矢量感，干净边缘，抗锯齿，专业企业级品质，16:9 横版比例
```

**风格要点**：
- 四色编码系统清晰区分不同概念
- 共享基座暗示统一框架
- 红色柱的发光效果强调其核心地位
- 图标简洁，一眼可识别含义
- 所有标签使用中文，"AI" 作为专业术语保留英文

## 示例 2：三阶段演进图（水平叙事流）

**来源**：Paradigm Shifts — 技术演进

**适用场景**：时间线、前后对比、技术演进、流程展示

**完整 Prompt**：
```
[PURPOSE]: 企业科技信息图配图，用于技术演进主题的演示文稿

[LANGUAGE]: 图片中所有文字使用中文，仅专业术语（如 AI、API 等）可保留英文

[STYLE]: 等距透视，扁平化设计+微立体感，企业科技风格，纯白背景，无面部特征简约人物（几何简化），柔和阴影

[COLOR PALETTE]: 色彩饱和度从左到右递增 - 左侧冷灰色系(#9CA3AF)，中间中性色调，右侧高饱和科技蓝(#3B82F6)，翡翠绿(#10B981)辅助，深色道路(#374151)

[COMPOSITION]: 从左到右的水平叙事流，三个阶段递进排列，底部有深色道路连接各阶段，白色虚线标记道路中心，人物/元素尺寸从左到右递增

[CONTENT]:
- 阶段 1（左侧，灰色调）：简约人物骑灰色马，代表传统/基础方法，整体低饱和度
- 阶段 2（中间，中性色调）：人物与未来感终端/亭子交互，亭子有屏幕显示AI标志，代表过渡/工具化阶段
- 阶段 3（右侧，高饱和蓝色调）：人物骑蓝色发光马，蓝色马暗示"增强现实"，代表优化/增强方案
- 连接元素：深色沥青道路，白色虚线，路旁有护栏和路标
- 背景元素：速度标志、信息路标等基础设施元素

[TEXT]:
- 总标题："范式转变"
- 阶段 1 标题："提示工程"
- 阶段 2 标题："工具化阶段"
- 阶段 3 标题："增强工程"

[QUALITY]: 高清矢量感，干净边缘，抗锯齿，专业企业级品质，16:9 横版比例
```

**风格要点**：
- 色彩饱和度递增是核心叙事手法
- 蓝色马（非自然色）暗示"增强/超越自然"
- 道路隐喻强化"行进/演进"概念
- 无面部人物最大化受众认同
- 所有阶段标题使用中文

## 示例 3：中心焦点封面图

**来源**：Cover — AI 控制与治理

**适用场景**：报告封面、白皮书封面、主题封面、核心概念展示

**完整 Prompt**：
```
[PURPOSE]: 企业科技信息图封面，用于AI控制/治理主题的报告或演示

[LANGUAGE]: 图片中所有文字使用中文，仅专业术语（如 AI、API 等）可保留英文

[STYLE]: 等距透视，3D渲染感，企业科技风格，纯白背景，柔和全局光照，半透明材质效果

[COLOR PALETTE]: 科技蓝(#3B82F6)主导（60%），珊瑚红(#E74C3C)强调（20%），翡翠绿(#27AE60)和琥珀黄(#F5A623)点缀（20%），白色基底

[COMPOSITION]: 中心焦点构图
- 核心：半透明蓝色发光大脑居中，放置在多层方形平台上（白色和蓝色交替）
- 环绕：六个护栏柱形成六角形围栏，连接彩色编码围栏带
- 氛围：漂浮的几何芯片元素（蓝/绿/黄/红色），淡色数据字符串作为背景
- 多层深度：前景围栏 → 中景平台+大脑 → 背景氛围元素

[CONTENT]:
- 中心大脑：半透明蓝色，可见白色神经通路纹理，柔和的青色辉光效果
- 平台：3-4层方形平台，白色和浅蓝色交替，象征技术栈的抽象层级
- 围栏带：红/绿/黄色编码，带二进制代码（0和1）图案，代表数据流和数字边界
- 氛围元素：小型几何芯片（三角形/菱形/方形），淡色数字字符串漂浮

[TEXT]:
- 主标题："AI 治理框架"
- 副标题："负责任的智能工程"

[QUALITY]: 高清3D渲染感，柔和全局光照，干净抗锯齿边缘，半透明材质，专业企业级品质
```

**风格要点**：
- 大脑+围栏的隐喻组合表达"受控的智能"
- 六角形围栏暗示多维度治理
- 二进制代码图案桥接人机可读性
- 半透明材质营造高级感
- 标题中"AI"作为专业术语保留英文，其余使用中文

## 示例 4：技术栈层叠图（层叠递进）

**适用场景**：技术架构展示、平台层次、基础设施说明

**完整 Prompt**：
```
[PURPOSE]: 企业科技信息图配图，用于技术架构/平台层次展示

[LANGUAGE]: 图片中所有文字使用中文，仅专业术语（如 AI、API、SaaS 等）可保留英文

[STYLE]: 等距透视，3D渲染感，企业科技风格，纯白背景，柔和全局光照

[COLOR PALETTE]:
- 底层：深灰色(#374151)，代表基础设施
- 中层：科技蓝(#3B82F6)，代表平台服务
- 上层：靛蓝(#6366F1)，代表应用层
- 连接元素：青色荧光(#00D4FF)

[COMPOSITION]: 层叠递进构图，从下到上三层结构
- 底层最大，向上逐层缩小
- 层与层之间有发光连接线
- 顶层放置核心图标或符号
- 侧面可见层级标签区域

[CONTENT]:
- 底层：深灰色实体平台，带电路板纹理，代表基础设施（计算/存储/网络）
- 中层：蓝色半透明平台，带数据流动效果（小光点移动），代表平台层（API/服务/中间件）
- 上层：靛蓝色发光平台，代表应用层（用户界面/业务逻辑）
- 连接：层间青色光柱/数据流，暗示数据在各层间流动
- 顶层：放置一个发光的齿轮或应用图标

[TEXT]:
- 总标题："技术架构层次"
- 底层标签："基础设施层（计算 / 存储 / 网络）"
- 中层标签："平台服务层（API / 中间件）"
- 顶层标签："应用层（用户界面 / 业务逻辑）"

[QUALITY]: 高清3D渲染感，柔和阴影，干净边缘，专业企业级品质，4:3 比例
```

**风格要点**：
- 层叠结构直观展示技术栈的抽象层级
- 层间光柱暗示数据流动
- 所有层级标签使用中文，API 等专业术语保留英文
- 侧面标签区域便于阅读各层名称

## 快速组合指南

需要生成新配图时，按以下步骤组合：

1. **确定构图** → 选择示例 1/2/3/4 的构图模式
2. **确定色彩** → 使用默认方案或选择色彩变体（见 prompt-templates.md）
3. **确定内容** → 从视觉隐喻库中选择合适的隐喻元素
4. **编写文字** → 将所有图片文字翻译为中文（专业术语除外），填入 `[TEXT]` 区块
5. **填充模板** → 将内容填入对应的 Prompt 模板
6. **调整比例** → 根据用途选择合适的 image_size
7. **生成并检查** → 使用质量检查清单验证结果，**重点检查图片文字是否为中文**
